Demo - Installation & Setup

In this demo, we will briefly showcase the installation of a lighting system using DALI devices and demonstrate how to adapt non-DALI components to work within the system.

Script

1. Show basic knowledge learning related to DALI

2. Show the DALI Lighting Lab, highlighting all included components (controller, connectors, wiring guides, etc.).

3. Do some physical installation process

4. Show how to configure the system through the configuration tool.

5. Operational test after configuration.
